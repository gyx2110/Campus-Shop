# 校园商铺平台

![Travis](https://img.shields.io/badge/language-Java-yellow.svg)<br/>

## 技术栈：

**前端**：前端UI库 SUI Mobile／ jQuery<br/>
**后端**：SSM／Spring Boot／图片开源工具 Thumbnailator / 验证码组件Kaptcha／对称加密算法 DES<br/>
**平台**：Web／微信测试号<br/>
**数据库**：MySQL/Redis<br/>
**部署**：DigitalOcean<br/>
